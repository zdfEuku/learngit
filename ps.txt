【1】选择 -》 色彩范围
快捷键：  ctrl + J  用选取新建图层 （通过拷贝的图层）
【2】窗口 -> 工作区 -> 复位基本功能
【3】文件：大小（长宽：像素单位或者厘米单位）
	   分辨率（单位： 像素/英尺）
	    颜色模式（RGB颜色 或者 CMYK颜色等）
【4】图层：
	图层类型
		文字
		形状
		智能对象
		调整图层
	可以按照 图层类型、名称等筛选
	图层不透明度
	链接图层：让几个图层同时移动
	ctrl + shift + N 新建图层
	ctrl + ]  或  ctrl + [    上下移动图层。 再加上shift置顶或底
	复制图层：  图层 -> 复制图层， 可以复制到其他文档
		    ctrl + J 或拖动到 新建图层按钮，  复制到当前文档
		    可以同时复制多个图层
